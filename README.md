# Information_security
Основы Криптографии

Задание 1. Анализ и классификация .
1. Веб-браузер устанавливает безопасное соединение с сайтом банка, видит зелёный замок - это ассиметричное шифрование, тк через него проходит установка защищенного соединения и обмен ключами.
2. Система проверяет, не был ли скачанный с интернета установочный файл повреждён или изменён злоумышленниками - это хеш-функция.
3. Пользователь входит на сайт, пароль которого хранится в базе данных в виде - тоэе хеш-функция.
4. Руководитель отправляет бухгалтеру электронный приказ о выплате премии. Бухгалтер должен быть уверен, что приказ действительно от руководителя и не изменялся - цифровая подпись.
5. Зашифрованная переписка в мессенджере (например, Signal или Telegram в режиме Secret Chat), где скорость важна - симметричное шифрование.

Задание 2. Расшифровка сообщения.
В задании указано, что при шифровании использовался сдвиг вперед, значит при расшифровке идетв обратном направлении.
1. Сначала повторяем алфавит русского языка (про букву "ё" снова все забыли)
2. После начинаем подбирать величину сдвига, сдвигая назад, пока не получим что-то похожее на осмысленное слово
3. При сдвиге на 14 позиций получается слово "игра", значит расшифровываем и остальные буквы используя сдвиг на 14
4. В итоге получаем "игра началась!"
5. Мы молодцы.

Задание 3. Криптографический пазл.
1. У Алисы есть два ключа (приватный и публичный)
2. Ей надо подписать сообщение: M="Заказ №451"
3. Для этого применяется хеш-функция к сообщ : H(M) = h1
4. Алиса шифрует только хеш, а не все сообщение
5. Она берет h1 и шифрует его приватным(1) ключом Pra. Получается S = Encrypt(h1(2); PrA(3))
6. Алиса отправляет Игорю сообщение и подпись (M,S)
7. Игорь получает пару и сам считает хеш полученного сообщ.  H(M) = h2.
8. Далее он расшифровывает подпись S с помощью публичного(4) ключа Алисы(5): h1' = Decrypt(S; PuA).
9. Далее имея два знач (h2, h1') он их сравнивает. Если h2 = h1' (6), значит всё корректно.
если хеши совпали, это означает, что сообщение не изменялось и что подпись создала именно Алиса, так как только она владеет приватным ключом.
